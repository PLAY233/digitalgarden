# MQTT主题说明
其中{SN}为设备蓝牙模块的IMEI，如terminal/l20/862057078551600/getInfo
## 平台侧订阅主题
遥测数据: terminal/l20/{SN}/getInfo
遥测数据主题协议格式包含以下协议内容
[[云洹网联/L20 V2协议#主动上报（网络式）\|#主动上报（网络式）]]中的设备信息
[[云洹网联/L20 V2协议#云洹自定义协议\|#云洹自定义协议]]中的遥测信息上报
[[云洹网联/L20 V2协议#标准运行识别信息协议\|#标准运行识别信息协议]]中的遥测信息上报

指令回复: terminal/l20/{SN}/callBack
指令回复主题主要是对平台下发的指令的回复，包含以下协议内容
[[云洹网联/L20 V2协议#设备配置\|#设备配置]]中的所有指令的返回数据及错误信息恢复

## 平台侧发送主题
指令下发: terminal/l20/{SN}/setInfo
指令下发主题主要是使用特定指令协议对设备进行配置的主题，包含以下协议内容
[[云洹网联/L20 V2协议#设备配置\|#设备配置]]中的所有指令的请求数据

# 指示灯状态说明
绿灯快闪：等待锁星
绿灯常亮：已锁星
蓝灯常灭：只蓝牙广播
蓝灯常亮：蓝牙+4G广播

# 设备配置
支持串口和MQTT配置
只有网络式上报支持配置协议类型
标准运行识别信息协议执行《GB4675-2025 民用无人驾驶航空器系统运行识别规范》
配置即时生效

| 字段                         | 类型     | 说明                                                                             |
| -------------------------- | ------ | ------------------------------------------------------------------------------ |
| method                     | string | 配置指令                                                                           |
| data                       | object |                                                                                |
| ->mqtt_server              | string | MQTT服务器地址及端口                                                                   |
| ->mqtt_tls                 | int    | 是否启用mqtts，mqtt服务器需要配置SSL证书，<font color="#c00000">不支持自签证书</font><br>0: 禁用，1: 启用 |
| ->mqtt_username            | string | MQTT 用户名, 默认：L20                                                               |
| ->mqtt_password            | string | MQTT 密码, 默认：123456                                                             |
| ->protocol_type            | int    | 网络式支持的协议类型<br>0: [[云洹网联/L20 V2协议#标准运行识别信息协议\|#标准运行识别信息协议]]，1: [[云洹网联/L20 V2协议#云洹自定义协议\|#云洹自定义协议]]                               |
| ->encrypt                  | int    | 是否开启加密                                                                         |
| ->apn                      | string | 物联网卡的APN, 此项无论是否设置都不影响手机卡                                                      |
| ->angle_compensation       | int    | 航向角补偿，此为磁力计补偿值，如实际角度为0，上报角度为90，则设置此值为-90                                       |
| ->uas_id                   | string | 唯一产品识别码(20个字符)                                                                 |
| ->reg_mark                 | string | 实名登记标志(8个字符)，参见[[云洹网联/L20 V2协议#数据内容项及编码取值要求\|#数据内容项及编码取值要求]]                                               |
| ->operating_category       | int    | 航空器系统运行类别，参见[[云洹网联/L20 V2协议#数据内容项及编码取值要求\|#数据内容项及编码取值要求]]                                                  |
| ->aircraft_category        | int    | 航空器分类，参见[[云洹网联/L20 V2协议#数据内容项及编码取值要求\|#数据内容项及编码取值要求]]                                                      |
| ->last_power_off_timestamp | int    | 上次关机时间，内部使用                                                                    |
| ->dismantle_status         | int    | 设备拆除状态                                                                         |

## FOTA升级
url 为差分包的http下载链接，升级前请确认固件包存在，否则设备不会进入升级流程。
固件命名规范：如 140T141.bin，代表V1.4.0版本与V1.4.1版本差分升级包，升级流程为 V1.4.0升级到 V1.4.1。
<font color="#c00000">注意：升级前请确认当前固件版本是否与差分包的起始版本一致。</font>

请求数据
```
{
  "method": "start fota",
  "data": {
    "url": "http://changyg.work:9000/l20/fota/140T141.bin"
  }
}
```

返回数据
```
{
  "method": "start fota response",
  "data": {
    "message": "success"
  }
}
```

# 文件系统指令
## 获取磁盘使用情况
获取C: D: 磁盘 总空间、已使用空间、剩余空间

请求数据
```
{
  "method": "get discSize",
  "data": {
    "disc": "D:"
  }
}
```

返回数据
```
{
  "method": "get discSize response",
  "data": {
    "total": "480M",
    "used": "368K",
    "free": "479M"
  }
}
```

## 获取目录信息
获取目录内所有文件及文件夹

请求数据
```
{
  "method": "get dirInfo",
  "data": {
    "path": "D:"
  }
}
```

返回数据
```
{
  "method": "get dirInfo response",
  "data": [
    {
      "name": "cache.dat",
      "type": 0,
      "size": 1024
    },
    {
      "name": "record",
      "type": 1,
      "size": 0
    }
  ]
}
```

## 读取文件
读取文件，一次不要读太多

请求数据
```
{
  "method": "read file",
  "data": {
    "path": "D:/cache.dat",
    "offset": 0,
    "size": 1024
  }
}
```

返回数据
```
{
  "method": "read file response",
  "data": {
    "size": 1024,
    "content": "base64编码后的文件内容"
  }
}
```

## 删除文件或文件夹
删除文件或文件夹

请求数据
```
{
  "method": "delete fileDir",

    "path": "D:/cache.dat"
  }
}
```

返回数据
```
{
  "method": "delete fileDir response",
  "data": {
    "message": "delete success"
  }
}
```
